如何使用edu邮箱享受学生权益

2021-09-17 学习记

  1. 1. 如何申请
  2. 2. 有什么用
  3. 3. 有什么福利
  4. 4. 学校拥有的其他正版资源

作为一名大学生，无论是使用专业软件的机会，还是和导师等交流的机会都变多了。这时有一个edu后缀的学校邮箱可以让你的沟通变得更加正式，并可以享受到各种学生的权益，如：JetBrains开发工具教育版的全套免费试用权等…目前，大多数大学都允许在校学生申请edu邮箱，本文的「如何申请」以BJUT为例，其他学校的申请方式可以咨询学校的信息中心。

本文参考：https://sspai.com/post/68227

如何申请

北京工业大学为每一位在校师生准备了教育邮箱，学生邮箱域名为@emails.bjut.edu.cn。

教育邮箱在个人的信息门户中注册。
注册流程：信息门户 –> 个人中心 –> 左侧栏“账户相关” –> 邮箱开户 –> 设置邮箱用户名及密码

注意邮箱的用户名一定是英文+数字等，不能是中文！（有同学设定了中文邮箱名字，根本无法登入，信息中心也无能为力）

登录网址：http://mail.bjut.edu.cn

有什么用

拥有一个edu后缀的邮箱，可以通过接受验证邮件来证明你的学生身份（有的平台会通过学信网验证码）。除去像普通邮箱一样可以正常收发信件，在BJUT校园网内可以当作一个免费的小云盘。

每个邮箱账户都有1G的免费云盘空间，如果在校内网环境下传输文件可以使用这个小云盘，可以避免一部分U盘拷贝带来的安全性问题。

有什么福利

  1. 苹果教育优惠

    拥有教育身份可以在购买iPad、Macbook、iMac等产品线时享受几百元不等的教育优惠。在暑期活动期间还有Airpods附赠，可以对比一下其他渠道的价格按需购入。

    同时，购买Apple的教育套装如：Logic Pro、Final Cut Pro X等，会便宜很多。

  2. 各种云服务

    拥有GitHub等账户并通过学生认证可以获得数项教育优惠。

    这一类优惠包含的软件众多，包括但不仅限于：

    1. Autodesk旗下的：AutoCAD、3DMAX、Maya等
    2. Jetbrains下的：IDEA、PyCharm、CLion等
    3. Adobe旗下的：Pr、Lr、Ps等
    4. 微软旗下的Azure云服务
    5. DigitalOcean VPS云服务器

    除此之外还有Google Workspace，微软的Office学生等

    各种云服务和软件在教育优惠的加持下都可以提供一个学生负担得起的价格，例如阿里云学生机（云高校计划）。

  3. 各种软件、订阅

    iPad下优秀的建模软件Shapr 3D在通过教育认证之后，可以获得免费的使用Pro版资格，省下一年888RMB的订阅费用。

    还有：

    1. OnTheHub教育商店
    2. Student App Centre
    3. 1Password密码管理软件
    4. Axure设计软件
    5. Affinity全家桶
    6. Spotify学生订阅
    7. Notion笔记、知识库、任务管理软件
    8. XFTP + XShell服务器管理软件

    等…

  4. 淘宝、京东、美团等

    淘宝/支付宝、京东、美团等也有相应的学生优惠，在购物或者订外卖的时候可以享受一部分折扣。

    拥有学生优惠的还有：

    1. 海底捞
    2. 爱奇艺
    3. 网易严选
    4. 迪士尼
    5. 顺丰快递

    等…

  5. 额外安利

    推荐一款互动视频项目「基本操作」，通过可视化和可交互技术把传统教科书改造为交互视频项目，帮助你直观清晰地理解复杂概念，感受科学之美，让你学到很多可能原先从未接触过的知识。

    网站是：https://jibencaozuo.com/

    同样可以使用学生优惠，享受课程半价，强烈推荐。

学校拥有的其他正版资源

学校为在校师生提供了众多软件及操作系统的正版化服务，通过按照正版化软件平台说明一步一步的操作便可以直接获取到价值上千的正版软件使用权。

微软正版化网址：http://ca.bjut.edu.cn ，里面提供了Office、Windows等微软正版软件。

MATLAB正版化软件平台：http://portal2nd.bjut.edu.cn/xxzx/cyrj/20181227/15458713922789257_1.html 使用教育邮箱可以登入MathWork的帐号，免费下载安装MATLAB建模软件。

这些软件从内网下载不会计入校园网流量。
