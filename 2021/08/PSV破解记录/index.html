PSV-2000破解记录

2021-08-18 折腾记

  1. 1. 破解之前
  2. 2. h-encore 破解
  3. 3. modoru 降级系统
  4. 4. 固化破解
  5. 5. NoNpDrm 插件安装
  6. 6. 黑商店安装
  7. 7. 写在最后

闲来无事，把好几年前在台湾买的索尼PSV-2000又翻了出来。上一次翻出来还是上个暑假，当时因为PlayStation Network（PSN）网络的帐号密码忘记了，导致PSV不能登录PS Store。以为这一台PSV已经死了，没想到经过一番操作之后又把PSN帐号找了回来，于是就有了机会破解一下这台PSV。

对于PSV的处理总共分成了几个不同的部分，我分开来记录。

破解之前

破解之前，先将记忆棒取出。在PS Store中登入PSN帐号，确认自己的系统软件版本再进行破解操作。

h-encore 破解

FinalHE的Github repo：https://github.com/soarqin/finalhe

PSV的破解我参考了这篇文章，感谢shipengliang对资料的汇总和总结。

  1. 按照文章的描述在电脑上面安装PSV的usb驱动，安装FinalHE破解软件，在PSV上面连接到电脑，勾选将h-encore瘦身到7MB。将h-encore和VitaShell软件导入到PSV主机上。

  2. 启动h-encore，按照软件提示进行破解操作。

    如果启动h-encore之后PSV黑屏，关机重启之后按住R肩键再启动h-encore。

  3. 在选择完安装HENkaku之后，退回系统设置，检查系统版本号后是否出现「变革」二字。如果出现，说明破解成功。

modoru 降级系统

modoru的Github repo：https://github.com/TheOfficialFloW/modoru

降级部分我参考了这篇文章，推荐把系统降级为3.65版本。

  1. 先在设置中的HENkaku设置里，取消启用PSN伪装和启用版本伪装的勾选。在自动启动设定中，取消所有勾选项。

  2. 下载并在PSV上安装modoru的vpk文件，vpk文件在VitaShell里面可以直接安装。

    打开VitaShell软件后，点击Start按钮可以开启文件传输模式，可以将电脑中的vpk文件拷贝到PSV上。

    PSV的固件可以从 https://darthsternie.net/ps-vita-firmwares/ 这里下载。

  3. 按照上述文章里面的描述将固件文件和文件夹放在指定位置。

    PSV开启飞行模式，重启PSV并确保PSV电量至少有50%以上电量。(否则modoru会自动关闭或禁止降级)

  4. 根据modoru软件里面的描述进行系统降级。

  5. 降级后在系统设置中的HENkaku设置里，将启用PSN伪装和启用版本伪装重新勾选。将版本伪装改为最新的PSV版本（目前最新版是3.73）

固化破解

固化破解部分我参考了这篇文章。

PSV不经过固化的话，每次重新启动系统之后，需要手动打开h-encore重新安装HENkaku。固化之后重启系统自动就安装了HENkaku，而且可以保证重置机器也不掉破解。

  1. 下载安装enso_ex

  2. 打开并根据软件提示进行系统固化

  3. 下载安装一键固化软件

  4. 打开并根据软件提示，把VitaShell固化到日历，把Vita工具箱固化到Welcome Park。

    如果提示权限不足，直接再固化一次就可以了。

  5. 此时可以把破解前取出的记忆棒重新插入。

    1. 插入原装记忆卡后，会提示：

      确定要将内置记忆卡的资料移动至外接式记忆卡吗?

      • 若移动资料，内置记忆卡中的资料会被删除。
      • 若不移动资料，资料将保留在内置记忆卡内，但插入外接式记忆卡时将无法使用内置记忆卡。

      选择否

    2. 之后会遇到升级系统的提示，点取消。

    3. 此时，系统会提示你：

      若要使用此记忆卡，需先更新系统软件。
      或者替此记忆卡格式化后；即可用于本主机。若要替记忆卡格式化，请进主页画面选择[设定] > [格式化] > [替记忆卡格式化]。

      按说明，去格式化下原装记忆卡即可正常使用原装记忆卡。

    4. 重新安装好之前未固化的APP，并用Vita工具箱装好NoNpDrm的3个插件，设定，HENkaku设置中的版本伪装改为最新的PSV版本（目前最新版是3.73）的即可。

  6. 重新启动PSV，检查系统版本应该有「变革」二字，即为固化成功。

NoNpDrm 插件安装

参考文章

由于一些PSV的破解游戏需要NoNpDrm，所以需要安装这个插件。

在已经安装好的Vita工具箱中，安装前三个插件，安装完最后一个插件之后重启PSV即可安装成功。

黑商店安装

参考文章

黑商店pkgj的Github repo：https://github.com/blastrock/pkgj/

由于很多PSV的破解游戏资源都在网盘上面，下载速度堪忧。安装黑商店之后，可以让PSV直接通过LiveArea下载游戏，只要是网络环境好，下载速度还是有一定的保障的。

  1. 在VitaShell中安装好pkgj之后，刷新一次列表。
  2. 出现刷新异常之后，将config.txt拷贝到指定目录下。
  3. 重新打开pkgj即可刷新出游戏列表，可以直接在里面下载软件。下载任务就自动添加到LiveArea里面开始下载了。

写在最后

我个人对于破解软件和盗版软件的态度是「不支持，不反对，不张扬」。

平心而论，使用破解和盗版软件是一个挺可耻的行为，而这篇文章讲的就是怎么进行这么无耻的行为（这么一想还是挺愧疚的？）。

大厂的情况还好一些。但是很多个人独立开发者就靠着卖软件挣钱，此时如果用户使用的是盗版软件，那岂不是断人财路吗。破解与反破解永远是魔高一尺道高一丈的较量。最后的结果就是开发者在与破解者的斗争当中耗尽精力，直到没有精力再去维护软件开发新功能，最后一个优秀的个人开发者就被盗版逼得没有退路。

诚然，有的时候确实是条件不允许我们使用正版软件，无论是网络条件还是自身的经济条件。但是使用盗版软件的时候应当心生愧疚，而不是大张旗鼓的告诉所有人，并且以使用盗版软件为荣，这种其实是最令人不齿的。

如果你觉得某个软件好，而且各种条件都允许的情况下，还是支持一下正版吧。
