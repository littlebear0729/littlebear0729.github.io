对于校园自行车管理的几点思考

2023-12-07 观

  1. 1. 政策内容是什么
  2. 2. 反对者为什么反对
  3. 3. 我的态度与看法
  4. 4. 校园自行车现状
  5. 5. 可以怎么办

前几日，学校出台了在校园内关于电动车管理的一系列新政策。作为一个入学以来，给校长信箱写过不下十篇邮件的「刺儿头」，还是斗胆写一篇博客谈一谈自己的看法。针对于这次的新政策，学生们的反应也十分多样。有支持的，也有反对的，主要能听到的还是反对的声音居多。我个人还是比较乐于看到学校愿意出手管这件事情的，对于一部分政策，我非常赞同。但是对于令一部分政策，我不敢苟同，认为还是有待斟酌。写下这篇文章供大家一起交流讨论。

政策内容是什么

2023年11月28日，校内网发了两篇通知，题目分别为《关于办理校园电动自行车备案的通知》和《关于在平乐园校区设置电动自行车专用停车场及规范车辆停放要求的通知》。

在《关于办理校园电动自行车备案的通知》中，要求对所有「通知发布日期之前」的「有购车发票或电动车行驶证」的电动车进行备案。在规定时间和过渡期过后，禁止未备案的电动自行车驶入校园。禁止所有电动自行车驶入中蓝校区。

在《关于在平乐园校区设置电动自行车专用停车场及规范车辆停放要求的通知》中，学校在校园内规定了两处电动自行车专用停放点和一处电动车专用充电处。要求「所有」电动自行车停放在两个指定停放点，并对于没有按要求停放的车辆「移动至」停车场且「后果自负」。在规定时间和过渡期过后，严格执行电动车集中停放。禁止所有电动自行车驶入中蓝校区。同时「倡议」师生员工在校园内不使用电动自行车作为通行工具。

反对者为什么反对

北工大可能是北京电动车保有量最大的高校之一了，学校面积不大，但是由于研究生宿舍在几公里外的中蓝校区，所以在两校区之间往返通勤的需求非常大。而购买电动车的大部分恰恰是有通勤需求的研究生。

在政策中，禁止了电动车进入中蓝校区，必然会加强中蓝校区周边道路的停车压力。于是，矛盾就将转移到市政部门与学生。这也是大多数反对者的主要顾虑。由于我是本科生，不便过多对中蓝校区的情况作出评价，后文不再讨论中蓝校区相关的内容。

另一个反对点在于，不允许将电动车作为校园内的通行工具有一些太过一刀切了。规定的停放地点也太远，结果就是大家都不是很方便。

我的态度与看法

在第一篇通知中，学校要求所有进入校园内的电动车备案。这样做便于学校对电动车进行管理，在电动车上牌的基础上进行备案，方便学校根据车牌号找到对应的车主。同时，这么做杜绝了校外电动车进入校园。对于备案政策，我是举双手赞成的。

在第二篇通知中，首先，学校「倡议」在校园内不使用电动自行车作为通行工具。大家都知道，学校的「倡议」基本上就是明令禁止。在《校园电动自行车安全使用承诺书》中，写着「本人承诺遵守不使用电动自行车作为校内通行工具」。说白了，学校不想让大家在校园内骑电动车，而是想让大家把电动车作为校外的代步工具。这一点依我来看是否有一些太过一刀切了？平乐园校区说大不大，说小不小。在有急事的情况下，电动车还是最方便的交通工具。我认为这点政策确实还有待商榷。

第二点是，学校要求所有电动车在指定的两个停车区停车。我认为这项政策也太过极端。首先这两个停车区的面积是否足够容纳所有的电动车还不得而知，而且这两个停车区的位置设定，距离大部分同学非常远。事实上，这项政策是遵循「电动车为校外代步工具」的假设前提下成立的。按照实际情况来看，学生们也有将电动车停放在教学楼下、食堂门口、办公楼下和宿舍楼门口的需求。如果在这些相应地方画好停车引导标志线，在不允许停车的地方画好黄色网格线。通过增加规定停车的位置，同学们可能会更容易接受这项政策，从而配合政策落地。其实在这里，最重要的点在于学校没有执法权。对于学生的电动车，谈不上「强制移动」，更谈不上「后果自负」。这一点也希望学校采用更温和的方式，对违规学生进行批评教育和处罚。

总得来说，学校如此制定政策，是实实在在的一刀切懒政，只想着用最低的成本解决问题。在制定政策之前没有广泛听取学生的意见，是本次政策制定的最大败笔，也是激起群愤的导火索。校方妄图通过一刀切的政策解决问题，殊不知这么做只会激化矛盾。认识问题、直面问题、解决问题是优秀管理的态度，无视问题、掩盖问题、回避问题是低水平的逃避。

我的另外看法还包括，目前的政策仅针对于电动自行车进行备案和管理，对于自行车的备案和管理还没有政策。我个人希望学校可以完善这方面的政策。另外，对于新购买的电动车如何进行上牌和备案，目前的政策是不允许进行备案。但是新学年一定会有这样的需求，希望学校可以继续完善相关政策。

校园自行车现状

同学，你见过被各种自行车围的水泄不通的宿舍楼门口吗？你见过停放了双层车辆，人都无法挤过去的食堂门口吗？你见过原本10米宽，但是两侧都停了自行车导致只有一人宽的道路吗？你有过被快速穿行的电动车、自行车刮蹭到的经历吗？这些我都经历过。除此之外，还有一些车辆经常停放在「热门区域」却从来没有移动过，座椅上积了厚厚的一层灰尘，成为了名副其实的「僵尸车」，阻碍了原本可以停放车辆的位置。

学校出台政策的背景绝对不只是应付消防安全检查，校方一定是看到了越来越多的非机动车在校园内的乱象，才出台政策准备解决这个问题。我结合个人的经历，也希望大家首先可以自觉将车辆停放在合理的位置，提升自己的安全骑行意识。在有条件的情况下，写信给学校，对自行车的管理提出自己的建议，配合施行学校指定的合理的政策。

总结一下，主要的问题在以下几点：

  1. 校园里自行车数量过多
  2. 僵尸车数量多
  3. 乱停放现象突出
  4. 骑车人安全意识淡薄

综上，究其根本原因，校园内的非机动车的矛盾点，从来就不是电动车和自行车数量的问题，而是非机动车车主没有自觉停放车辆、安全骑行意识低下的问题。这个问题与电动车还是自行车无关，与学生的道德、素质水平、自觉性，和学校的管理水平、管理决心有关。

可以怎么办

针对于上文总结的主要问题，我提出如下建议：

  1. 充分规划停车位，建设遮雨棚、自行车架，划定明显标志，请同学们规范停放车辆。
  2. 在特定地点修建凸面镜、画定自行车道等交通安全设施。
  3. 增强对同学们的安全骑行教育。
  4. 处理僵尸车。
  5. 进行车辆编号和登记（即备案制度）。
  6. 建立完善的惩罚制度。

就近期学校颁布的政策来看，目前的结果恐怕是大家都不想看到的。学校不愿意花钱修建遮雨棚，只愿意使用警戒线围住楼门口。不舍得投资修建便利设施，永远无法解决问题。我只能说，学校的管理水平还需要再学习一个。如果在学校规定的过渡期之后，校园内自行车问题得不到改善，我将继续给校长信箱写信提出以上建议。

希望学校可以以正面引导、批评教育为主，惩罚为辅，拿出解决问题的态度，共同建立美好的校园环境。

最后，附上我在2023年10月23日给校长信箱的信件全文内容如下：

校长信箱，

您好！

目前，学校内的自行车和电动自行车的数量急剧增多。加之共享单车的使用量较大，校园自行车管理成为较大挑战。

在图书馆东侧与美食园相邻的道路上，经管楼门口，10号、11号宿舍楼门口均可看到大量随意停放的自行车和电动车。这些自行车的随意停放大量占用了行人通道，甚至占用了无障碍通道，为师生带来诸多不便。在宿舍楼门口堆积的自行车更会引发安全问题。一旦宿舍楼失火，楼内学生无法通过门口逃生，甚至消防通道也会被自行车占用，造成的后果不容小觑。

随意骑行、超速骑行的自行车还会带来安全问题。有的驾驶员不礼让行人，在校园路口（银杏大道与图书馆南门的十字路口）不观察左右来车和行人，极易引发事故。[1]

我建议学校可以加强对学生的自行车骑行安全教育，规定同学们有序、规范停放自行车。

建议在学校内划分充足的合理的非机动车指定停放点并设立醒目的指示牌和区域标识。规定共享单车、自行车和电动自行车停放在指定停放点（北操场北侧、金工楼南侧等），标明停放自行车的具体区域，禁止学生双层停车。同时对于这些停放点，适当修建遮雨棚，使学生们停车更加放心。

建议对所有进入校园的自行车做一车一人的登记，将车辆管理编号粘贴在车辆醒目位置，方便入校时安保人员进行查验。对没有停放在规定区域的自行车根据管理编号找到学生进行批评教育和处罚，对于没有管理编号的“僵尸车”进行统一清理。

希望在不久的将来可以看到校园内摆放整齐的自行车，共同维护文明有序的校园交通环境。

祝好，

[署名]

*注1：我的确有一次差点被后座载人的电动车撞到。骑车人在前面扭头和后面的人说话，没有看路。我甚至站定在路中间没有走动，看着他们说笑，径直撞向我。气得我大叫了一句：看着点！还有一次与一辆快速骑行的自行车擦肩而过，对方的书包甚至撞到了我的胳膊上。

只能说，学校听了，但是只听了一半，听了最方便做的那一半。
